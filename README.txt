JADOO OM ONLINE-APP

Diese Dateien sind bereit für Netlify, GitHub Pages, Cloudflare Pages oder jeden normalen Webserver.

Schnellste Veröffentlichung mit Netlify Drop:
1. Den gesamten Ordner Jadoo-OM-Online-App öffnen.
2. Alle Dateien/Unterordner als ZIP hochladen oder den Ordner am Computer auf Netlify Drop ziehen.
3. Die erzeugte https://...netlify.app-Adresse in Safari öffnen.
4. iPhone: Teilen > Zum Home-Bildschirm.
5. Android: Browser-Menü > App installieren.

Wichtig: Die Audiodateien stammen aus der vom Nutzer bereitgestellten Datei. Rechte zur öffentlichen Verbreitung bitte selbst prüfen.
